[Git] window에서 git 간단하게 사용하기

1. Git Bash 실행하기.
     바탕화면 우클릭 Git Bash Here

2. Git 환경 설정하기.
     참고 URL : https://git-scm.com/book/en/v2/Getting-Started-First-Time-Git-Setup
     $
     $ git config --global user.name "parksh"
     $ git config --global user.email "mzncvmc@gmail.com"
     $ git config --list

3. Git Bash 간단명령어 소개.
     폴더이동, 폴더생성, txt 파일 생성
     $
     $ cd /D
     $ mkdir GIT
     $ cd GIT
     $ echo "test001" > t1.txt
     $ echo "test002" > t2.txt
     $ ls -al

4. 본격적으로 git 사용하기
 1) 현재 폴더를 git 로컬 저장소로 등록하기
     git init 을 실행하게 되면, 경로명 뒤에 (master)가 붙는다.
     $
     $ cd /D/GIT
     $ git init
     $ ls -al

 2) 로컬 저장장소 확인하기
     빨간글씨로 .txt 파일이 추적되지 않은 파일(untracked files)라고 나온다.
     commit 하기 전에 index에 등록 한 후, commit를 해야한다.
     $
     $ git status
햣 

 3) 파일이 없는 경우
     만약 해당 폴더에 아무 파일이 없다면 아래 그림과 같이 나온다.(3. Git Bash 간단명령어 소개.  참조!)
     하지만 필자는 윗단에서 .txt 파일 두개를 만들었으니 생략...ㅋ
     $
     $ git status

 4) index 설정하기.
     개별 index 지정할때는 "git add 파일명" 폴더 안에 모든 파일 index 설정은 "git add ."
     모든 파일 index 설정시 제외파일 확장명 설정은 .gitignore 파일 내부에 설정 하면 된다.
     $
     $ git add t1.txt
     $ git status
     $ git add . 
     $ git status

 5) commit 하기.
     로컬 저장소에 commit 한다.
     $
     $ git commit -m 'initial commit'
     $ git status

5. 원격 저장소 관리하기
 1) 원격저장소 확인하기
     원격저장소가 연결된적이 없어서 아무것도 나오지 않는다.
     $
     $ git remote -v

 2) 원격저장소 추가하기
     아래 방법으로 실행 하고 github 홈페이지에서 git_test 원격 저장소를 만드는것을 추천한다.
     참조 URL : https://github.com
     $
     $ git remote add origin https://github.com/sungho7777/git_test.git
     $ git remote -v

 3) 원격저장소에 Push 하기(즉. 업로드!!)
     $
     $ git push origin master
     $ git remote show origin

 4) Commit Github에서 확인하기.
     URL : https://github.com/sungho7777/git_test

 5) 새파일 생성 부터 등록까지...마지막에 확인!



6. clone 만들기
     다른 사람이 git commit 한 프로젝트를 로컬 저장소에 다운로드 함.
     clone 할 URL 주소가 https://github.com/jayu108/book_parallel_programming_with_python 인경우...!
     URL 주소 뒤에 폴더명을 주어도 됨.
     $ 
     $ cd /D/GIT
     $ git clone https://github.com/jayu108/book_parallel_programming_with_python.git
     $ ls -al
